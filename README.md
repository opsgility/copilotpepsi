# Microsoft 365 Copilot Workshop — G&J Pepsi

Welcome to the G&J Pepsi Copilot Workshop. This full-day hands-on session covers Microsoft 365 Copilot across every major app — Copilot Chat, Word, Excel, Outlook, Teams, PowerPoint, and Copilot Studio Agents — using scenarios built for your business.

## Getting Started

Complete these three steps before the first exercise. For the full setup instructions including the prompting framework, see the **[Lab Setup](GJ-Course/00-lab-setup.md)**.

### Step 1: Download the resource files

1. Select the link below to download the resource files:

    **[Download ResourceFiles.zip](https://github.com/opsgility/copilot-pepsi/raw/refs/heads/main/GJ-ResourceFiles/ResourceFiles.zip)**

2. Once the download completes, open **File Explorer** and navigate to your **Downloads** folder.
3. Right-click **ResourceFiles.zip** and select **Extract All...**.
4. Choose a location (your **Downloads** folder or **Desktop** is fine) and select **Extract**.
5. After extraction, you should see a folder containing nine files (two `.xlsx` and seven `.docx`).

### Step 2: Upload the files to OneDrive

1. Open **Microsoft Edge** and navigate to [https://www.office.com](https://www.office.com).
2. Sign in with your **Microsoft 365** credentials.
3. Select **Apps** in the left navigation pane, then select **OneDrive**.
4. In OneDrive, select **+ New** in the top-left corner, then select **Folder**. Name it **GJ-Workshop** and press **Enter**.
5. Open the **GJ-Workshop** folder you just created.
6. Select **+ New** > **Files upload** (or drag and drop the files directly into the browser window).
7. In the File Explorer window that appears, navigate to the folder where you extracted the zip file. Select **all nine files** and then select **Open**.
8. Wait for the upload to complete. You should see a confirmation message: **Uploaded 9 items**.
9. Verify that all nine files appear in your **GJ-Workshop** folder:
    - GJ_Q4_Sales_Performance.xlsx
    - GJ_Warehouse_Ops_Report.xlsx
    - GJ_New_Hire_Onboarding_Guide.docx
    - GJ_Annual_Benefits_Summary.docx
    - GJ_Quarterly_Business_Review.docx
    - GJ_Route_Optimization_Proposal.docx
    - GJ_IT_Infrastructure_Upgrade_Plan.docx
    - GJ_Leadership_Meeting_Transcript.docx
    - GJ_Supply_Chain_Review_Transcript.docx

### Step 3: Add files to your Most Recently Used list

Copilot in some apps can only reference files from your **Most Recently Used (MRU)** list. Open these files so Copilot can find them during the exercises:

1. In OneDrive, select **GJ_Q4_Sales_Performance.xlsx** to open it in Excel for the web. Once it loads, close the tab.
2. Select **GJ_Quarterly_Business_Review.docx** to open it in Word for the web. Once it loads, close the tab.
3. Select **GJ_New_Hire_Onboarding_Guide.docx** to open it in Word for the web. Once it loads, close the tab.

> **Important:** Microsoft 365 Copilot can only work with files saved to OneDrive. Files stored locally on your PC must be uploaded to OneDrive for Copilot to access them.

You're ready to begin the exercises.
